# README

## Backend

It runs on port 3000 or 5000

### Steps

Run `npm i` to install all the dependencies

Then run `npm start` to run the project.

## Frontend

Runs on port 3000 by default, but offers to use another
one if 3000 is not available.

### Steps

Run `npm i` to install all the dependencies

Then run `npm start` to run the project
